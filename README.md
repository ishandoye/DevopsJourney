<details>
<summary> Use AWS CLI </summary>

- Install AWS CLI
  [AWS CLI Installation Guide](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html)
- Configure AWS on the local device
 - Run the command below to access your AWS account using your user.
	<pre>
	$ aws configure</pre>
- Manage Access Keys
  - [How to Manage Access Key](https://docs.aws.amazon.com/IAM/latest/UserGuide/id_credentials_access-keys.html)
    - Do not use the root user; instead, use a normal user with root privileges and create an access key. Save it for future use, but keep it private.
    - Use that access key in the ```aws configure``` command with your suitable region and the format like json.
